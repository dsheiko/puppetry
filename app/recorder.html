Welcome to Puppetry Recorder

Puppetry Recorder we use to record user interactions on the target web pages as a test suite. To start you need to type in the toolbar input URL of a page to navigate and press Enter. As pages loaded you can interact with them (click elements, type in inputs, move mouse cursor, resize window, scroll, navigate pages and so on).

Available tools

  • Address bar - navigate to the provided URL
  • "Select device" - resize window to match the selected device screen
  • Ctrl-Shift-S - add "make a screenshot" command to the log. Can be used to mark a particular step in your session
  • Recording - stop/continue recording
  • OK - save recording as a test suite and return to Puppetry main window
